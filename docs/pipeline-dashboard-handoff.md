can be embedded via a small canvas overlay.

- Loop seamlessly for idle state animations
- Provide both idle (subtle pulse) and active (intense glow) variants if possible

  Primary: Cyan #22D3EE
  Secondary: Teal #14B8A6
  Success: Emerald #34D399
  Warning: Amber #FBBF24
  Error: Red #F87171
- All animations must loop seamlessly
- Style: sci-fi industrial, holographic data visualization, glowing energy effects
- No solid fills — use gradients, glows, particles, and light streaks

GENERATE THESE 9 ANIMATION ASSETS:

ASSET 1 — "Knowledge Drop Overlay" (Stage 1: Intake Funnel)
- Glowing digital file icons (3 types: document, PDF, image) gently floating downward

- A bright cyan ring pulses at the funnel mouth, expanding outward like a ripple
- A numerical badge in the top-right glows and increments (placeholder: "247")

- Mood: Calm intake, organized flow

- Multiple interlocking gears (3-4) rotating with teal edge glow
- Tiny bright sparks fly off gear teeth at contact points

- A progress bar at the bottom fills from 0% to 100% with cyan, then resets
- Loop duration: 6 seconds
- Mood: Industrial processing, mechanical precision

ASSET 3 — "Radar Search Overlay" (Stage 3: Vector Retrieval)
- A bright cyan radar sweep line rotating 360° from a central point
- As the sweep passes over positions in the void, small orbs (8-12) light up briefly
- 3-4 orbs are "selected" — they glow brighter and drift toward center with trailing light
- A similarity score displays: "0.87" in cyan monospace font
- Loop duration: 5 seconds
- Mood: Discovery, precision targeting

ASSET 4A — "Confidence Gate GREEN" (Stage 4: High Confidence)
- A circular green lens flares brilliantly (emerald #34D399)
- A gauge needle swings smoothly from left to right, landing in the green zone

- Loop duration: 3 seconds

ASSET 4B — "Confidence Gate RED" (Stage 4: Low Confidence)
- A circular red lens flares (#F87171)

ASSET 5 — "Live Data Feeds Overlay" (Stage 5: Satellite Array)

- 5-6 random icons pulse violently with bright cyan at staggered intervals
- Remaining icons stay dimly lit (30% opacity)
- Thin cyan signal lines beam downward from active icons toward a central point
- Loop duration: 8 seconds
- Mood: Real-time intelligence, signal reception

ASSET 6 — "Assembly Line Overlay" (Stage 6: Prompt Construction)

ASSET 7 — "Delivery Overlay" (Stage 7: Response Output)

- 2-3 small badges ("Source A", "Source B") pop in around the bubble with spring animation
- The bubble and badges glow briefly then settle to a steady soft glow
- A filing cabinet icon at bottom-right glows amber briefly (save indicator)
- Loop duration: 5 seconds
- Mood: Completion, delivery, satisfaction

ASSET 8 — "S-Curve Data Pulse" (Path Animation)

- Loop duration: 8 seconds
- Mood: Continuous data flow, pipeline heartbeat

OUTPUT FORMAT: Provide each asset as a Lottie JSON animation definition.
If Lottie is not possible, provide as animated SVG with embedded CSS @keyframes.
Each asset must have a transparent background and be designed to overlay
on a dark (#0a0a0a) background.

## What You Do

### Step 1: Generate Animation Assets with Gemini 2.5 Pro Preview (Nano Banana)

The complete Nano Banana prompt is inside `pipeline-dashboard-handoff.md` under the section "NANO BANANA PROMPT". Feed that entire prompt to Gemini. ### Step 2: Measure the Base Plate

Open `rag-factory-baseplate.jpg`. Measure or estimate where each of the 7 machines sits as percentage positions (left %, top %, width %, height %) relative to the full image. Approximate positions are in the handoff doc but you should refine them by looking at the actual image.

### Step 3: Build the Dashboard Page

Replace `web/dashboard/src/app/page.tsx` with a new animated pipeline dashboard. The handoff doc has the complete React component structure, but here's the core concept:

- Base plate image as responsive background (`position: relative`, `aspect-ratio` from actual image dimensions)
- 7 clickable overlay zones positioned over each machine, each linking to the relevant dashboard page:
  - Stage 1 (Knowledge Drop) → `/knowledge`
  - Stage 2 (Factory) → `/knowledge`
  - Stage 3 (Question) → `/agents?tab=test`
  - Stage 4 (Confidence Gate) → `/workshop`
  - Stage 5 (Live Data) → `/settings`
  - Stage 6 (Assembly) → `/agents`
  - Stage 7 (Delivery) → `/agents?tab=test`
- Nano Banana animation assets overlaid on each zone
- Live metric badges on each zone pulling from existing API endpoints (details in handoff doc)
- An S-curve data pulse animation traveling the pipe path
- An amber research indicator that appears when auto-research tasks are pending
- Mobile fallback (below `lg:` breakpoint): vertical card list with same data and links, no image

### Step 4: Add CSS

Add the glow keyframes from the handoff doc to `web/dashboard/src/app/globals.css`.

### Step 5: Test

The dev server runs at `localhost:3002` from `web/dashboard/`. Run `npm run dev` if it's not already running. Verify:
- All 7 zones are clickable and navigate to the correct pages
- Live metrics update (vector count, file count, conversations)
- Animations play and are aligned with the machines in the base plate
- Mobile view shows the card fallback
